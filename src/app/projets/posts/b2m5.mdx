## Objectifs de cette mission

- Approfondir la programmation orientée objet (POO) en Java, en utilisant ArrayList pour manipuler des collections de données dynamiques.
- Travailler avec des API externes (NASA) pour récupérer et filtrer des données.
- Comprendre et pratiquer les tests unitaires (TU) et les tests fonctionnels (TF).
- Concevoir une interface graphique en Java Swing pour afficher les données traitées.
