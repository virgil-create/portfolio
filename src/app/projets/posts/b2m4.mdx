## Objectifs de cette mission

- Manipulation des API et bibliothèques Java
- Gestion des fichiers et exceptions
- Accès aux ressources du projet avec File et gestion des chemins.
- Gestion des exceptions spécifiques et utilisation de Logger pour le suivi des erreurs.
- Implémentation d'algorithmes pour :
        Recherche exacte dans un tableau.
        Recherche de mots contenant des critères spécifiques.
- Extraction et retour de sous-ensembles de données dans un tableau.
- Utilisation de méthodes de la classe String comme contains() et startsWith().
- Documentation et organisation du code :
        Écrire des Javadoc pour décrire les méthodes et leurs paramètres.
        Structuration d'une classe Java avec un constructeur et des méthodes bien définies.
