## Objectifs de cette mission

- Créer un répertoire local et cloner le projet sur Git.
- Configurer une branche Git personnelle et effectuer des modifications initiales (README.md).
- Exécuter et comprendre les tests unitaires existants (classe DepartementTest).
- Écrire les tests unitaires pour deux nouvelles classes : RegionTest et CommuneTest.
- Exécuter et analyser la classe Main.java pour comprendre son objectif.
- Étudier et expliquer la classe DAOGeo ainsi que la différence entre classe métier et classe technique.
- Comprendre l’utilité des fichiers du répertoire data et des scripts Python associés.
- Réfléchir sur le choix de technologies : pourquoi des scripts Python plutôt que Java ?
