## Objectifs de cette mission

- Comprendre comment générer un Bearer Token sur GitLab auto-hébergé.
- Utiliser ce token pour interagir avec l'API GitLab via Postman.
- Tester différents endpoints GitLab (récupération d'informations utilisateur, gestion de projets, etc.).
- Écrire des tests automatisés dans Postman.
